正文

wget下载:

安装： tar -zxvf nginx-1.6.2.tar.gz

进行configure配置：

编译安装 make && make install

启动Nginx：

启动命令：

/usr/local/nginx/sbin/nginx -s start 关闭（stop）重启（reload）

成功：

查看是否启动（netstat -ano | grep 80）

失败：
